# Настройка каналов

## Описание

**Канал** - это точка назначения куда можно отправить конечный результат работы из [модуля](Modules.md). Каналы настраиваются в конфигурационном файле, в разделе **notifications.channels**, так же каждый канал можно настроить в отдельном файле, положив его в папку указанную в ключе **notifications.path**. Количество каналов не ограничено, но у каждого нового канала должно быть уникальное имя.

Реализованные типы выходов:

- [Telegram](channels/TELEGRAM.md)
- [Email](channels/EMAIL.md)
